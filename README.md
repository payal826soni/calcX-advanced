# Advanced Calculator

A modern and feature-rich calculator built using **HTML, CSS, and JavaScript**. This project goes beyond basic arithmetic by integrating scientific functions, programmer tools, unit conversions, graph plotting, themes, history tracking, and an interactive user interface.

The goal of this project is to provide an all-in-one calculator experience while demonstrating advanced front-end development concepts and JavaScript programming techniques.

---

## Overview

Advanced Calculator is a responsive web application designed to handle a wide range of calculations through multiple specialized modes.

The application includes:

- Standard Calculator
- Scientific Calculator
- Programmer Calculator
- Unit Converter
- Graph Plotter

It also features theme customization, calculation history, angle mode switching, quick utility tools, and a modern dashboard-inspired interface.

---

## Features

### Multiple Calculator Modes

#### Standard Mode

Perform everyday arithmetic operations including:

- Addition
- Subtraction
- Multiplication
- Division
- Percentage calculations
- Decimal operations

---

#### Scientific Mode

Advanced mathematical functions including:

- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Square Root
- Logarithm (log)
- Natural Logarithm (ln)
- Power Operations
- Factorial
- Pi (π)
- Euler's Number (e)
- Auto Parenthesis Balancing

---

#### Programmer Mode

Programmer tools for number system conversion:

- Binary (BIN)
- Octal (OCT)
- Decimal (DEC)
- Hexadecimal (HEX)

Real-time conversion between all supported number systems.

---

#### Unit Converter

Supports conversion between:

### Length

- Meter
- Kilometer
- Centimeter
- Millimeter

### Weight

- Kilogram
- Gram
- Pound

### Temperature

- Celsius
- Fahrenheit
- Kelvin

#### Graph Plotter

Visualize mathematical functions directly inside the calculator.

Examples:

## How to Use

### Standard Calculations

Enter numbers and operators using the calculator keypad.

---

### Scientific Functions

Switch to Scientific Mode and use functions such as:

```text
sin()
cos()
tan()
sqrt()
log()
ln()
```

---

### Programmer Mode

Convert values instantly between:

```text
BIN → OCT → DEC → HEX
```

---

### Unit Conversion

1. Select a category.
2. Choose source unit.
3. Choose target unit.
4. Enter a value.
5. View the converted result instantly.

---

### Graph Plotting

1. Switch to Graph Mode.
2. Enter a mathematical expression.

Example:

```javascript
Math.sin(x)
```

3. Click **Plot**.

The graph will be rendered on the canvas.
